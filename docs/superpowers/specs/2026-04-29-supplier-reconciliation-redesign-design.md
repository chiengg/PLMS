# 供应商对账页面改版设计

日期：2026-04-29

## 背景与目标

对“供应商对账”页面做结构重排与功能扩展：

- 将“日账单 / 月账单”一级 Tabs 移动到查询条件区域上方
- 日/月底部增加二级 Tabs（采购对账单 / 采样对账单）
- 新增筛选项“1688账号（下拉单选）”
- 移除“生成对账单/生成账单”按钮与相关逻辑
- 列表支持“详情/下载”；“详情”进入路由二级页展示明细字段

## 范围

### In Scope

- 一级页（日账单/月账单）结构重排与筛选项调整
- 二级 Tabs（采购对账单/采样对账单）与列表展示
- 二级详情页（路由页面）与字段展示（采购对账单详情、采样对账单详情）

### Out of Scope

- 实际对接后端接口（本项目继续使用 mock 数据与本地状态）
- 下载文件真实生成（保留 UI/交互与占位行为）
- 权限控制与账号体系真实联动

## 交互与信息架构

### 一级结构

页面从上到下：

1. 一级 Tabs：日账单 / 月账单（默认：日账单）
2. 查询筛选区（位于一级 Tabs 下方）
3. 二级 Tabs：采购对账单 / 采样对账单（默认：采购对账单；日/月一致）
4. 列表区：汇总表（字段一致；日/月的“账期”显示格式不同）

### 查询筛选区

字段：

- 账单时间（必有）
  - 日账单：年月日区间（Input type=date，start/end）
  - 月账单：年月区间（Input type=month，start/end）
- 供应商（下拉）
- 采购平台（下拉）
- 采购员（下拉）
- 1688账号（新增：下拉单选）
- 按钮：搜索 / 重置

移除：

- “生成对账单/生成账单”按钮及弹框逻辑

### 二级 Tabs 与列表

二级 Tabs：

- 采购对账单（purchase）
- 采样对账单（sample）

列表字段（两类一致）：

- 账期
- 供应商
- 采购平台
- 采购员
- 1688账号
- 创建日期
- 操作（详情、下载）

账期显示格式：

- 日账单：YYYY-MM-DD
- 月账单：YYYY-MM

## 二级页（路由页面）

### 路由结构

二级页面采用路由方式打开（支持浏览器返回、可复制链接）。

建议路由：

`/supplier-reconciliation/:cycle/:type/:id`

- cycle: daily | monthly
- type: purchase | sample
- id: 列表行 id

## 组件拆分建议

推荐拆分为可复用组件，便于后续继续扩展筛选项与字段：

- `ReconciliationCycleTabs`：日账单/月账单
- `ReconciliationFilters`：筛选区（含账单时间与 1688账号）
- `StatementTypeTabs`：采购对账单/采样对账单
- `StatementTable`：汇总列表（根据 cycle/type 选择数据源与列标题）
- `StatementDetailPage`：二级路由页（根据 type 渲染不同字段）

## 数据与状态（前端）

- `cycle`：daily | monthly（一级 Tabs）
- `type`：purchase | sample（二级 Tabs）
- `filters`：
  - `billStart` / `billEnd`（日账单 date / 月账单 month）
  - `supplier`
  - `platform`
  - `purchaser`
  - `account1688`
- `rows`：mock 数据列表（按 cycle/type 区分或同一结构不同内容）

## 验收标准

- 一级 Tabs 位于筛选区上方，默认选中“日账单”
- 日/月筛选区均有“账单时间”：
  - 日：年月日区间
  - 月：年月区间
- 筛选区新增“1688账号”下拉单选
- 不再显示“生成对账单/生成账单”按钮
- 筛选区下方存在二级 Tabs：采购对账单/采样对账单（默认采购对账单；日/月一致）
- 列表字段为：账期、供应商、采购平台、采购员、1688账号、创建日期、操作（详情/下载）
- 账期格式：日=YYYY-MM-DD，月=YYYY-MM
- 点击“详情”跳转路由二级页：
  - purchase：展示采购对账单详情字段
  - sample：展示采样对账单详情字段

## 追加需求（V2：账单金额与全供应商明细列表）

### 一级列表（汇总表）新增字段

- 新增列：账单金额
- 位置：在“1688账号”右侧、“创建日期”左侧
- 金额口径：
  - 采购对账单：账单金额 = 应付金额合计（示例数据）
  - 采样对账单：账单金额 = 总金额合计（示例数据）

### 示例数据规则（一级列表）

- 日账单：采购对账单、采样对账单列表均提供“全部供应商”的汇总数据（供应商列显示“全部”）
- 月账单：采购对账单、采样对账单列表均提供“全部供应商”的汇总数据（供应商列显示“全部”）

### 详情页改造（从表单网格 → 列表表格）

详情页不再仅展示单条表单字段，改为“明细列表表格”展示多行数据：

- 日账单详情：展示“当日所有供应商”的账单详情列表
- 月账单详情：展示“当月每日 × 所有供应商”的账单详情列表

为满足“所有供应商”与“当月每日”的展示维度，详情表格额外增加两列：

- 供应商（建议置于最左侧）
- 账单日期（月账单详情需要；建议置于供应商左/右相邻位置）
